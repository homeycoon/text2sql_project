# Text2SQL Service

Сервис, который превращает ваши вопросы на естественном языке в SQL-запросы и возвращает готовые данные.  
Работает на **FastAPI** + **LangGraph** + **GigaChat**, умеет кэшировать схему БД и выгружать результаты в CSV, Excel или Markdown.

## Как работает сервис

- Вы вводите свой запрос на естественном языке; 
- На основе вашего запроса и схемы БД с помощью LLM генерируется SQL-запрос;
- SQL-запрос валидируется и выполняется;
- Полученный результат экспортируется как файл.

## Особенности

- **Модель LLM** — сервис написан под `GigaChat`, можно использовать любую из их доступных моделей (указать в `.env`).
- **Валидация и безопасность** — сгенерированный запрос проверяется на наличие запретных операций.
- **Кэширование схемы БД** в Redis (обновляется раз в час).
- **Экспорт результатов** — доступен в форматы CSV, Excel, Markdown (LLM сам выбирает нужный по вашему запросу).
- **Готов к запуску через Docker Compose** — PostgreSQL, Redis и приложение в связке. БД наполняется тестовыми данными.

## Архитектура

```mermaid
graph LR
    A(Начало) --> B[db_schema_node<br>Получение схемы БД]
    B --> C[sql_generate_node<br>Генерация SQL через GigaChat]
    C --> D{Ошибка?}
    D -->|Да| E(Конец)
    D -->|Нет| F[sql_exec_node<br>Выполнение и валидация]
    F --> G{Ошибка?}
    G -->|Да (ошибка)| C
    G -->|Нет| H[export_node<br>Экспорт результата]
    H --> E
```

Внутри графа четыре узла:
1. **db_schema_node** — получение схемы (с кэшем).
2. **sql_generate_node** — генерация SQL через GigaChat.
3. **sql_exec_node** — валидация и выполнение запроса.
4. **export_node** — форматирование результата (CSV/Excel/MD).

## 🚀 Быстрый старт

### 1. Клонируйте репозиторий
```bash
git clone <url>
cd text2sql-service
```

### 2. Настройте переменные окружения
Скопируйте пример `.env.docker.example` в `.env.docker` и укажите свои **GigaChat credentials** и параметры БД.

### 3. Запустите Docker Compose
```bash
docker-compose up --build -d
```

Сервис будет доступен по адресу [http://localhost:8000](http://localhost:8000).

## Использование

### 🔹 Веб-форма
Откройте главную страницу, введите запрос (например, *«покажи топ-3 отдела по средней зарплате сотрудников»*) и получите файл с результатами.

Ответ — файл (CSV по умолчанию; можно попросить в формате Excel или Markdown).
